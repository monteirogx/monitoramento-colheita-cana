Projeto: Sistema de Gestão - Cana-de-açúcar

Solução focada na redução de perdas na colheita mecanizada de cana-de-açúcar. O sistema monitora a produtividade por talhão, calcula perdas reais e emite alertas críticos caso a perda ultrapasse 15%, auxiliando na tomada de decisão imediata sobre a regulagem das colheitadeiras.
